## 위 모든 항목 완료 후 → Claude Code에게 전달

아래 내용을 Claude Code에 전달하면 Phase 1 개발을 시작합니다:

```
준비 완료됐어. Phase 1 시작해줘.

SUPABASE_URL: https://xxx.supabase.co
SUPABASE_ANON_KEY: eyJ...
번들 ID: com.nc.midnightpostoffice
기획 미결 항목 결정:
  1번: (선택)
  2번: (선택)
  3번: (선택)
  4번: (선택)
  5번: (선택)
  6번: (선택)
```

> service_role 키, Firebase JSON 같은 민감한 키는 대화창에 직접 붙여넣지 말고
> `.env` 파일과 `supabase secrets set` 명령어로 직접 처리하세요.
