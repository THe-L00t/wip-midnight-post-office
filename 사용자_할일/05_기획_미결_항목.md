## 1. 수신자를 어떻게 찾나?

편지를 보낼 때 수신자를 어떻게 지정하는지 결정이 필요합니다.

**선택지:**

| 방식 | 설명 | 장점 | 단점 |
|------|------|------|------|
| A. 닉네임 검색 | 닉네임 입력 후 검색 | 직관적 | 스팸 가능성 |
| B. 우체국 코드 | 각 우체국마다 고유 코드 발급 | 의도적 공유만 가능 | 불편할 수 있음 |
| C. 중앙우체국 통해서만 | 모르는 사람에게는 중앙우체국 경유 | 게임 컨셉에 맞음 | 진입 장벽 있음 |
| D. 혼합 (B + C) | 코드로 직접 or 중앙우체국 경유 | 유연함 | 구현 복잡도 증가 |

## 3. 경유자가 편지 내용을 볼 수 있는지

편지가 내 우체국을 거쳐갈 때, 내용을 볼 수 있는지 여부입니다.
게임 감성과 프라이버시 모두에 영향을 미칩니다.

**선택지:**

| 방식 | 설명 |
|------|------|
| A. 볼 수 없음 | 봉인된 편지, 경유자는 내용 모름 |
| B. 일부만 공개 | 첫 줄 또는 제목만 보임 |
| C. 완전 공개 | 경유자도 내용 전체 열람 가능 |
